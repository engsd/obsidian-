## 核心概念

- 我们现在有了强大的军事力量（肌肉）和为其供血的财政系统（循环系统）。但要让这两者高效协同，还需要一个强大的中央处理器和神经网络——一个前所未有的、渗透到社会每个角落的官僚国家（骨架与神经）。
