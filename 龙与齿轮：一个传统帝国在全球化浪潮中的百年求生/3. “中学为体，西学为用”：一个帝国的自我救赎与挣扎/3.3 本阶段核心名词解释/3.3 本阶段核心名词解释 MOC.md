- 在经历了外部的暴力冲击和内部的太平天国动乱后，清帝国的一部分精英阶层终于意识到“不变不行”。于是，一场以“师夷长技以制夷”为目标的自救运动开始了。这是清帝国第一次主动、系统地对全球资本主义体系作出回应。
