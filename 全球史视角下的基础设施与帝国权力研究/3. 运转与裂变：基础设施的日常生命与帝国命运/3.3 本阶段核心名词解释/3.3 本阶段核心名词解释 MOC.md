# 3.3 本阶段核心名词解释

## 核心概念

- 基础设施建成后并非故事的终点，而是其权力效应真正开始弥漫的起点。它的日常运转维系着帝国，而其故障与废弃则预示着帝国的裂痕。

- [[3.3.2 生命周期：指一个产品、系统或工程从设计、建造、运营、维护到最终废弃或回收的完整过程。]]
