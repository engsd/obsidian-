## 核心概念

- 我们已经掌握了复杂系统的核心理论，但这还不够。科学需要工具来验证和探索。本章将介绍两大利器——网络科学和主体建模，它们如同思想的望远镜和实验室，让我们能看见、能模拟那些看不见的系统结构和动态演化。
