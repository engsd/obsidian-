# 🤖 Python自动化脚本 - 核心功能说明

> **注意：本README专门介绍项目中的Python自动化脚本，其他文件夹仅为示例内容**

## 🎯 核心脚本概述

本项目包含多个强大的Python自动化脚本，它们构成了知识库构建的核心引擎。这些脚本实现了从Markdown大纲到完整Obsidian知识库的自动化构建。

## 📋 主要脚本文件

### 1. `全自动知识库构建器_修正版.py` - 主构建引擎

#### 🔧 核心功能
- **智能大纲解析** - 自动识别Markdown的层级结构
- **文件夹结构生成** - 根据标题层级创建对应的文件夹
- **原子笔记创建** - 自动生成标准化的笔记模板
- **MOC（内容地图）生成** - 创建主题导航笔记
- **知识钩子提取** - 自动提取思考问题和链接
- **统计报告生成** - 输出构建过程的详细统计

#### 🎨 特色功能
- **BOM字符处理** - 修正了UTF-8 BOM导致的解析问题
- **错误恢复机制** - 增强的文件读取稳定性和错误提示
- **状态变量重置** - 确保模块解析的正确性
- **链接优化** - 优化的内部链接和输出格式

#### 📝 使用语法
```markdown
## 主题名称           → 创建顶级文件夹
*核心概念描述*        → 斜体内容写入MOC核心概念
### 子主题名称        → 创建子文件夹
- 原子笔记标题        → 创建原子笔记
- 钩子: 思考问题      → 添加知识钩子到对应笔记
```

### 2. `create_structure.py` - 结构创建器

#### 🔧 核心功能
- **基础结构创建** - 根据大纲创建文件夹和文件结构
- **文件名清理** - 自动移除Windows非法字符
- **MOC笔记维护** - 自动更新内容地图笔记
- **进度统计** - 显示创建过程的详细统计

### 3. `通用大纲转Obsidian.py` - 格式转换器

#### 🔧 核心功能
- **格式标准化** - 将通用大纲转换为Obsidian格式
- **模板生成** - 创建标准化的笔记模板
- **批量处理** - 支持批量转换多个文件

## 🚀 使用方法

### 基本用法
```bash
# 自动检测当前目录的.md文件并构建
python 全自动知识库构建器_修正版.py

# 指定特定的大纲文件
python 全自动知识库构建器_修正版.py 我的大纲.md

# 指定文件和输出目录
python 全自动知识库构建器_修正版.py 我的大纲.md ./输出目录
```

### 帮助信息
```bash
python 全自动知识库构建器_修正版.py -h
python 全自动知识库构建器_修正版.py --help
```

## 🎨 输出结构

脚本会自动生成以下结构：
```
主题名称/
├── 子主题1/
│   ├── 子主题1 MOC.md      # 内容地图笔记
│   ├── 原子笔记1.md        # 具体内容笔记
│   └── 原子笔记2.md
├── 子主题2/
│   ├── 子主题2 MOC.md
│   └── 原子笔记3.md
└── 主题名称学习总览.md      # 总览导航
```

## 🔗 链接体系

脚本建立的三级链接体系：
1. **原子笔记** → 包含具体内容和知识钩子
2. **MOC笔记** → 主题内容地图，链接所有相关原子笔记
3. **学习总览** → 整个知识领域的导航中心
