## 核心概念

- 年鉴学派掀起的革命浪潮至今仍在激荡。它不仅深刻改变了历史学本身，其影响力更溢出到其他学科。然而，任何伟大的范式都会有其局限性。我们需要批判性地审视它的遗产，并展望历史学未来的新方向。
